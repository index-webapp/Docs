SURCHI provides real-time on-chain analytics across seven major blockchain networks, with new chains added regularly. Each supported network is fully indexed — meaning SURCHI tracks every token, swap, liquidity event, and wallet interaction from genesis to the present — and all seven chains are queryable from the same search interface without any manual network switching. The sections below describe each network's technical details and the specific SURCHI features available for it.

#### Pump.fun Integration

SURCHI tracks every token launched on Pump.fun in real time. Pump.fun is a Solana-native launchpad that enables permissionless token creation using an automated bonding curve mechanism, making it one of the most active sources of new token issuance in crypto.

SURCHI's Pump.fun integration covers the full token lifecycle:

- **Launch detection** — New Pump.fun token creations are indexed within seconds of the on-chain event, appearing immediately in the SURCHI trending feed and search index.
- **Bonding curve status** — SURCHI displays the current position of each token on its Pump.fun bonding curve, including the percentage of the curve filled and the USD value of SOL deposited.
- **Graduation detection** — When a Pump.fun token reaches the end of its bonding curve and qualifies for migration, SURCHI detects the graduation event and updates the token's status in real time.
- **Raydium migration tracking** — After graduation, SURCHI monitors the on-chain migration transaction that moves the token's liquidity from the Pump.fun bonding curve to a Raydium AMM pool, and automatically links the new pool to the existing token page so your analytics continue uninterrupted.
- **Sniper and early buyer analysis** — SURCHI identifies wallets that purchased within the first few transactions of a Pump.fun launch, flagging potential coordinated sniper activity in the holder analysis view.

|---|---|

| Native Token | ETH |

| Avg Block Time | ~2 seconds |

**Indexed DEXes:** Uniswap V3, Aerodrome Finance

Base is an Ethereum Layer 2 network built on the OP Stack and incubated by Coinbase. Base has seen rapid growth in DeFi activity, particularly through Aerodrome Finance, which became the dominant DEX on the network shortly after launch.

**SURCHI features for Base:**

- L2 transaction cost display in ETH and USD

- Base-native token launch monitoring with rapid indexing optimized for the network's active memecoin scene
