## Real-Time Blockchain Indexing

SURCHI indexes multiple blockchains simultaneously through a distributed ingestion layer that subscribes to each network's latest block data as it is produced. New blocks are parsed, decoded, and stored within milliseconds of finalization, meaning the data you see in the SURCHI Explorer reflects the current state of each chain — not a stale snapshot from minutes ago.

- **Indexing latency:** Sub-second for Solana and Arbitrum; 1–3 seconds for Ethereum and BNB Chain; near-instant for Base and Avalanche.
- **Historical depth:** Full transaction and event history from each chain's genesis block is available for querying.
- **WebSocket streaming:** SURCHI exposes live WebSocket feeds so you can receive push notifications the moment a transaction, liquidity event, or price movement occurs — no polling required.
- **Parallel chain coverage:** All seven supported networks are indexed concurrently, so switching from an Ethereum token to a Solana token never introduces a context-loading delay.

## Market Analytics

For every token SURCHI tracks, a full set of market analytics is computed and refreshed continuously:

- **Market Capitalization** — Circulating supply multiplied by current price, updated on every new trade.
- **Fully Diluted Valuation (FDV)** — Total supply multiplied by current price, giving you a ceiling valuation for tokenomics analysis.
- **Total Liquidity** — Aggregated liquidity depth summed across every indexed DEX pool for that token.
- **24-Hour Volume** — Rolling 24-hour swap volume in USD, segmented by buy and sell pressure.
- **Transaction Count** — Total number of on-chain swaps, transfers, and contract interactions within configurable time windows.
- **Price Impact Estimates** — Projected slippage for common trade sizes based on real-time pool depth data.
