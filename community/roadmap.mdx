SURCHI's mission is to become the most comprehensive, intelligent, and accessible multi-chain analytics platform in the blockchain ecosystem. Getting there is a deliberate journey — from a focused Solana-native launch to a global, multi-chain intelligence layer serving individual traders, developers, and institutions alike. The roadmap below outlines the five phases of that journey, including what has already been delivered, what is actively in development, and where the platform is heading next.

## Roadmap Timeline

<Steps>
  <Step title="Phase 1 — Foundation (Completed)">
    Phase 1 established the core infrastructure and brought the $SURCHI token and initial platform to market. Every deliverable in this phase has been completed.

    - **$SURCHI token deployment** — the native token was deployed on Solana using the SPL token standard, with a fixed total supply of 19,897,905 tokens. The mint authority was revoked after deployment to guarantee supply immutability.
    - **Presale and liquidity pool creation** — 60% of the total supply (11,938,743 tokens) was allocated to the presale, and 30% (5,969,372 tokens) was paired with SOL to seed the initial liquidity pool on a Solana DEX.
    - **Core Explorer launch (Solana)** — the SURCHI Explorer went live with full support for Solana tokens, including real-time price data, volume metrics, holder counts, and transaction history.
    - **Basic token analytics and holder data** — the platform launched with token detail pages covering price charts, top holders, market cap, fully diluted valuation (FDV), and liquidity depth.
    - **Website and documentation launch** — surchi.xyz and this documentation portal launched alongside the platform, providing users with guides, API references, and community resources.
  </Step>
  <Step title="Phase 2 — Platform Build (In Progress)">
    Phase 2 is focused on building out the platform's core intelligence features and expanding chain coverage beyond Solana. Several deliverables are live; others are in active development.

    - **AI Contract Audit engine (Solana)** — the AI-powered audit tool is live for Solana tokens, analyzing smart contract code and on-chain behavior to produce risk scores, honeypot detection results, and detailed security breakdowns.
    - **Wallet Intelligence dashboard** — the Wallet Intelligence feature is live, enabling users to analyze any on-chain wallet's trading history, PnL performance, token holdings, and behavioral patterns.
    - **Multi-chain expansion: Ethereum & BNB Chain** — SURCHI is extending Explorer and AI Audit support to Ethereum mainnet and BNB Chain, significantly broadening the platform's addressable token universe.
    - **Real-time price alerts** — a configurable alert system allows users to set price, volume, and holder-change notifications for any tracked token, delivered via browser notifications and email.
    - **API v1 release (beta)** — the SURCHI REST API is available in beta, giving developers programmatic access to token data, audit scores, wallet analytics, and market metrics.
    - **Premium tier with $SURCHI gating** — a premium subscription tier is being introduced, unlocking higher API rate limits, advanced analytics, and priority audit processing for users who hold or stake a minimum $SURCHI balance.
  </Step>
  <Step title="Phase 3 — Expansion (Upcoming)">
    Phase 3 broadens SURCHI's reach to all major EVM-compatible chains and non-EVM ecosystems, introduces cross-chain liquidity tooling, and delivers the developer SDK and mobile experience that users have been requesting.

    - **Full multi-chain support** — complete Explorer, AI Audit, and Wallet Intelligence coverage for Polygon, Base, Avalanche, and Arbitrum, bringing the total number of supported networks to seven.
    - **Smart Liquidity Routing (cross-DEX)** — an intelligent routing engine that identifies the optimal swap path across multiple DEXs and chains to minimize slippage and maximize execution quality.
    - **SDK release (JavaScript + Python)** — officially supported client libraries for JavaScript/TypeScript and Python, making it easy for developers to integrate SURCHI data into their own applications and bots.
    - **Webhook system** — a real-time event delivery system that pushes token price changes, new holders, large transactions, and audit flag updates directly to developer-specified endpoints.
    - **Mobile app (iOS + Android)** — a native mobile application for both iOS and Android platforms, bringing the full SURCHI experience — including portfolio tracking, alerts, and wallet analysis — to your pocket.
    - **Pump.fun real-time launchpad tracker** — a dedicated dashboard for tracking newly launched tokens on Pump.fun in real time, complete with bonding curve progress, holder velocity, and early AI risk scoring.
  </Step>
  <Step title="Phase 4 — Intelligence Layer (Planned)">
    Phase 4 transforms SURCHI from an analytics platform into a predictive intelligence layer, introducing advanced AI capabilities, institutional-grade data products, and full DAO governance.

    - **Advanced AI: on-chain behavior prediction** — next-generation machine learning models that analyze historical on-chain patterns to generate probability-weighted predictions for token price trends, liquidity events, and wallet behavior.
    - **Smart Money wallet feeds** — curated, real-time feeds of transactions from algorithmically identified high-performance wallets (so-called "smart money"), enabling users to follow the moves of consistently profitable on-chain traders.
    - **Cross-chain portfolio tracking** — a unified portfolio dashboard that aggregates token holdings and PnL across all supported chains into a single view, with cross-chain cost-basis accounting.
    - **DeFi yield analytics** — coverage of major DeFi protocols including lending platforms, liquidity pools, and yield aggregators, with APY tracking, impermanent loss calculations, and risk-adjusted return metrics.
    - **Enterprise API plans** — dedicated high-throughput API tiers with SLA guarantees, dedicated infrastructure, custom data feeds, and priority support for institutional and commercial customers.
    - **SURCHI DAO formation** — the transition to a fully on-chain DAO with smart contract-enforced execution, a multi-sig community treasury, and a community-elected governance council. See the [Governance page](/community/governance) for details.
  </Step>
  <Step title="Phase 5 — Ecosystem (Vision)">
    Phase 5 represents SURCHI's long-term vision: an open, composable data ecosystem where third-party developers and data partners can build on top of the SURCHI platform and contribute to a growing intelligence network.

    - **SURCHI data marketplace** — a permissioned marketplace where data contributors — including node operators, independent researchers, and analytics teams — can publish and monetize on-chain datasets, with $SURCHI used for access payments and contributor rewards.
    - **Third-party plugin ecosystem** — an open plugin framework allowing community developers to build and distribute custom analytics modules, chart types, alert conditions, and AI models directly within the SURCHI interface.
    - **Cross-chain bridge integration** — native bridge functionality within the SURCHI platform, enabling users to move assets across chains without leaving the analytics environment.
    - **Institutional data partnerships** — formal data-sharing agreements with institutional research firms, data aggregators, and blockchain infrastructure providers to enhance data quality and coverage.
    - **Additional chain support** — planned expansion to emerging Layer 1 networks including Sui, Aptos, and TON, ensuring SURCHI remains relevant as the multi-chain landscape evolves.
  </Step>
</Steps>
